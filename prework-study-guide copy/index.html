HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element allows you to implement an image from an outside source or file.
  • HTML is how you structure a website usuing key elements to allow you make a easy to understand frame for a website.

CCS

  • The purpose of CCS is to style a website to make it more fun to use
  • using elements in an external css file you style each aspect of the page in different ways
  • there are multiple ways to create color in a css file (color: blue) would change the intended elemnts blue
  • A website is set up in boxes, it is your job to adjust the boxes using css
  • The margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A math function helps us select a random number from the const.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array and log them to the console.
  • An Array is a single variable that is used to hold a group of data.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
